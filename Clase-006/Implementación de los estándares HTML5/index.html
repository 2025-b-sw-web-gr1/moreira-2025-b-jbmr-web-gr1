Implementación con HTML5 y CSS3

Esta página es un ejemplo práctico donde se aplican los estándares HTML5 y CSS3 definidos por el W3C. Se utilizan etiquetas semánticas, diseño responsivo y Flexbox para organizar el contenido de forma clara y accesible.

Características principales

HTML5 semántico

Uso de etiquetas como <header>, <nav>, <main>, <section>, <article> y <footer> para describir mejor la estructura del documento.

CSS3 y Flexbox

El layout de las tarjetas y la barra de navegación se construye con Flexbox, facilitando la adaptación del diseño a distintos tamaños de pantalla.

Diseño responsivo

Se utilizan unidades relativas y media queries para que la interfaz se vea correctamente en dispositivos móviles, tablets y escritorios.

Contenido principal
