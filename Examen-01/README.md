# Examen-01 — Documentación Swagger (OpenAPI)

Este examen documenta los endpoints utilizados en la práctica de Bruno (Clase 008-009) usando el estándar **OpenAPI 3.0** (Swagger).

## API Base
- https://jsonplaceholder.typicode.com

## Archivo principal
- `openapi.yaml`

## Cómo visualizar
1. Abrir https://editor.swagger.io/
2. Copiar el contenido de `openapi.yaml` en el editor
3. Revisar y probar endpoints desde Swagger UI
